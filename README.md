# Echo
A local AI chatbot built with Node.js, Express, Ollama, and Discord integration.

## Features
- Browser-based chatbot UI
- Terminal chatbot version
- Shared chat logic through `services/chat.mjs`
- Discord slash command integration
- Per-user conversation memory in Discord
- Persistent conversation history saved to file
